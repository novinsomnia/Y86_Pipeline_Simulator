1.	程序运行系统：OSX
	需安装：jdk 1.7.0

2.	从终端(Terminal)打开至含有pipeline.jar的文件夹
	输入命令 java -XstartOnFirstThread -jar pipeline.jar
	程序开始运行

3.	按键说明：
	load		载入文件 *.yo
	save		导出当前输入文件所对应的每个时钟周期的流水线寄存器的值
	frequency滑动条	调节运行频率
	run/stop	开始/暂停运行
	back/next	单步后退/前进
	set		跳转至jump to cycle文本框中所输入的周期
	reset		重置
	check memory	查看起始地址(from address)起所输入长度(length)位的存储器内容
